# PrismEval

> Build trustworthy AI by measuring it.

PrismEval is an open-source AI evaluation and observability platform for Large Language Models (LLMs), Retrieval-Augmented Generation (RAG) systems, and AI agents.

Instead of building another chatbot, PrismEval helps developers measure, monitor, and improve AI applications by evaluating response quality, latency, cost, hallucinations, relevance, faithfulness, and semantic drift.

## Vision

Modern AI applications need reliable evaluation and observability.

PrismEval provides developers with a centralized platform to collect AI traces, evaluate outputs using automated metrics and judge models, visualize performance, and identify issues before they reach users.
